About FERS

FERS (Festival Event Registration System), is a free service provided by the New Codington Chamber of Commerce, for all that live in the city of New Codington. The services include the user to register to events in the local area via the website rather than using a congested telephone service. We aim to provide the best rates and best matched events suited towards you.


Please submit any questions, comments or concerns about the usage, registration, event availability, or any other information presented on this website to us, if required.



Please note and be aware that any events/activities that are scheduled via the Festival Event Registration System are subject to postponement and cancellation at any time. Please frequently check the site for latest events are ticket availability.
